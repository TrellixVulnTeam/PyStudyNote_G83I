1.严格使用缩进，tab或4个空格，两者不能混用
2.表达式括号是可选的,建议省略: if x < y <==> if (x < y)
3.结束一行代码无需分号";",直接换行即可
4.一行中可以执行多条语句，用';"间隔，不建议这样做,这是python中唯一需要分号的地方
5.一个语句可以占用多行(用括号括起,或者方括号,大括号)

Python语法规则:
    语句是逐个运行的,从头到尾,除非你不这样写(流程控制跳跃)
    块和语句的边界会自动检查
    复合语句 = 首行 + ": " + 缩进语句
    空白行,空格即注释(#字符开头)通常会被忽略
    文档字符串会忽略,但会保存并有工具显示
语句的分隔符:
    如果使用语法括号(),[],{}对,语句就可横跨数行
    如果语句以反斜线结尾,就可以横跨数行
    字符串常量有特殊规则: 三重引号字符串块可以横跨数行,相邻字符串常量隐式相加
